# WishList
Miniprojekt 2. semester (Efterår 2025). Tech: Spring boot, Thymeleaf, JPA, PostgreSQL.

Spring boot webapp hvor brugere kan oprette, vise og dele ønskesedler. 
Projektet indeholder CI/CD, Qodana, tests, ER-diagram, klassediagram og dokumentation. 

Struktur :
- src/main/java – kode (controller, model, repo, service)
- src/main/resources/templates – Thymeleaf HTML
- src/main/resources/sql – SQL scripts
- docs/ – diagrammer og dokumentation

Mvp: 
Regristrer/bruger via email
Opret ønskeseddel
Tilføj ønsker (navn, pris, link, beskrivelse)
Delingslink

Dokumentation:
ER diagram: docs/er-diagrams.md
Klassediagram: docs/class-diagram.md
